# 04-Journal

Time-anchored writing: daily notes (`YYYY/MM/`), plus the three subfolders below. This
is `config.taxonomy.digests_dir` by default; the judge's searchable daily digest (one
section per day, in a monthly note) also lands directly under here.

- `Decisions/`: one note per significant decision, with the reasoning behind it.
- `Weekly/`: weekly lint + digest output (autonomous) and, later, curated newsletters.
- `gate-queue/`: escalation queue ONLY (see `CLAUDE.md`). Items here wait for the owner;
  everything else is written straight to canon by the deterministic appliers.
