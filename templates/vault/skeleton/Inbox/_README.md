# Inbox

Quick-capture buffer. Anything goes here first when there is no time to file it
properly: a raw idea, a pasted transcript, a link. Not canon: retrieval and queries skip
this folder for knowledge, but the owner can still ask "what's pending in Inbox".

Process regularly (a vault-organizer pass once it grows past roughly 10 notes) rather
than letting it become a second, unstructured vault.
